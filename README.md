# Collection, a theme for Zenphoto
**Collection** is a responsive theme for Zenphoto image gallery and CMS. 

## Supported plugins
* cacheManager
* class-video
* comment_form
* contact_form
* favoritesHandler (*no Multiple sets in v.1*)
* menu_manager
* openstreetmap
* rating
* register_user
* tag_suggesr
* themeSwitcher
* user_login_out
* zenpage

## CSS+
* prefers-color-scheme
* svg icon for locked galleries
